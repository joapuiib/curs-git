    Bloc 4: Git avançat¶

    En aquest bloc es treballaran operacions més avançades de Git.

    Etiquetes: Creació d'etiquetes per identificar versions.

    Reserva de canvis (stash): Guardar canvis de manera temporal.

    Reset: Eliminar commits de la història.

    Commit amend: Modificar l'últim commit.

    Revert: Desfer un commit sense eliminar-lo de la història.

    Cherry-pick: Aplicar un commit d'una branca a una altra.

    Merge squash: Fusionar una branca amb un sol commit.

    Exercici: Git avançat: Exercici pràctic d'aquest bloc.
