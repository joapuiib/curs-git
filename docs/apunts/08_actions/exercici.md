*[PR]: Pull Request

## Objectius
Els objectius d'aquest exercici són:

- Conéixer com configurar fluxos de treball automatitzats amb GitHub Actions.
- Entendre el funcionament bàsic d’un flux de treball (esdeveniments, jobs i passos).
- Adaptar un flux de treball a les necessitats d’un projecte concret.


## Lliurament
Per a lliurar aquest exercici heu d’indicar el següent a la bústia de AULES:

- L'enllaç al repositori de :simple-github: GitHub.
- Una breu descripció del flux de treball creat:
  - Quin esdeveniment el dispara (push, PR, etc.).
  - Quina tasca automatitza.
  - Quin resultat produeix.

El flux de treball s'ha d'haver executat almenys una vegada al repositori per a demostrar que funciona correctament.


## Enunciat
En aquesta tasca, heu de crear un repositori de :simple-github: GitHub
i automatitzar una tasca relacionada amb aquest.

Podeu triar entre qualsevol dels [[actions-exemples]] o bé investigar i crear una automatització pròpia.

Es recomana que automatitzeu alguna tasca d’un projecte que treballeu a l’aula o que siga d’interés personal, com ara:

- Execució automàtica de proves.
- Verificació d’estil de codi.
- Generació automàtica de documentació.
- Desplegament en un entorn de proves.
